Create a modern, clean, scalable web UI design for a health-tech AI platform named "Avenution".

Avenution is a food recommendation system based on user's body condition using AI.

Design Goals:
- Clean and minimal
- Modern SaaS startup feel
- Scalable design system
- Professional but friendly
- Desktop-first
- Support both Light Mode and Dark Mode

Color System:
Primary color: Deep Rose Red (#C62828)
Accent: Soft Green (#16A34A)
Light mode background: #F9FAFB
Dark mode background: #0F172A
Use neutral grays for structure.
Avoid overly bright medical red.

Typography:
Modern sans-serif (Inter or Poppins)
Strong hierarchy and clean spacing
Rounded corners (medium radius)
Soft shadows

Pages to design:

1. Landing Page
- Clean navigation (Logo: Avenution, Features, Technology, Login)
- Hero section with headline:
  "Smart Food Recommendations Based on Your Body Condition"
- CTA buttons:
  - Try Now (Primary)
  - Login (Secondary)
- 3 feature cards:
  - Body Condition Analysis
  - Personalized Menu Recommendation
  - Progress Tracking
- Simple modern footer

2. Guest System Page (Main Feature)
- Centered input form card
- Body condition inputs (age, weight, blood pressure, etc.)
- Submit button
- Result section below with recommendation cards
- Subtle CTA after result:
  "Save your history? Login to track your progress."

3. User Dashboard (Login Required)
- Sidebar navigation (Dashboard, History, Profile, Logout)
- Greeting header
- Quick Check card
- Recommendation history list
- Clean data cards

4. Admin Dashboard
- User table
- Data summary cards
- Simple analytics section

Design it using modern SaaS layout system.
Make spacing breathable and structured.
Ensure the design can scale into a full product.
Include Light and Dark mode variations.
